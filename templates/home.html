Welcome, we are group 46, DEVISUALS, and this is our project for the course DBL. You can build your own visualizations by uploading a dataset by pressing the upload button and then head to the visualization page. We have 5 visualizations to choose from: Bar Chart, Scatter Plot , Heat Map and a Gaze Strip Plot (you can click on these links as well). At the current state of the webtool it is not possible to select Everyone when using the Gaze Strip Plot, so please keep that in mind. If a memory error occurs while using the Gaze Strip Plot then head back to the Scatter plot and change the previously selected user to another one and try again. Thank you for visiting and hope you enjoy our webtool!
